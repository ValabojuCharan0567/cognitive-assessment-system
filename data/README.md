# Local data (optional)

This folder is for **small, local-only** sample files you do not want under version control (see root `.gitignore`).

- **Large / research datasets** should live outside the repo and be pointed to with `DATASET_PATH` (default `~/Datasets/CognitiveAssessment`). See the main [README.md](../README.md).
- Use `data/` for quick experiments: tiny WAV snippets, CSV exports, or notes—nothing required for the API to run.
